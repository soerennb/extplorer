# User Stories

## General User
*   **As a user**, I want to **browse files and directories** on the server so that I can locate specific items.
*   **As a user**, I want to **create new files and directories** to organize my data.
*   **As a user**, I want to **upload files** from my local machine to the server.
*   **As a user**, I want to **download files** from the server to my local machine.
*   **As a user**, I want to **edit text files** directly in the browser with syntax highlighting so I can make quick code changes.
*   **As a user**, I want to **copy, move, rename, and delete** files and folders to manage the file structure.
*   **As a user**, I want to **change file permissions (chmod)** to secure my files.
*   **As a user**, I want to **search for files** by name or content to find lost items.
*   **As a user**, I want to **compress files into archives (Zip, Tar, etc.)** and **extract existing archives** to manage storage space and transfer groups of files.
*   **As a user**, I want to **view file details** (size, date, permissions) to understand the file status.

## Administrator
*   **As an admin**, I want to **manage users**, including creating, updating, and deleting accounts.
*   **As an admin**, I want to **assign home directories** to users to restrict their access to specific areas.
*   **As an admin**, I want to **reset user passwords** if they are forgotten.
*   **As an admin**, I want to **view system information** to monitor the server environment.

## Remote User
*   **As a remote user**, I want to **connect via FTP or SSH2** so I can manage files on remote servers using the same interface.
*   **As a user**, I want to **access files via WebDAV** so I can mount the remote file system locally on my OS.
